# История изменений

## [1.0.8] — 2026-03-20
- 🔒 Безопасность: права Unix-сокета изменены с `0o666` на `0o660`
- 🔒 Безопасность: устранена race condition (TOCTOU) при создании сокета
- 🔒 Безопасность: добавлена валидация `gw_name` против path traversal (только `[a-zA-Z0-9_-]`)
- 🔒 Безопасность: добавлена проверка формата UUID перед передачей в shell-команды
- 🔒 Безопасность: `shell_exec("kill -0 ...")` заменён на `posix_kill()`
- 🔒 Безопасность: добавлена блокировка файла при работе с `/tmp/gateways.status`
- 🔒 Безопасность: добавлен `htmlspecialchars()` для описаний интерфейсов в API
